Six rarity tiers in order from lowest to highest:
⚪ Common — 55% drop chance — grey colour
🟢 Uncommon — 25% drop chance — green colour
🔵 Rare — 10% drop chance — blue colour
🟣 Epic — 7% drop chance — purple colour
🔴 Mythic — 2.5% drop chance — blood red colour
🟡 Legendary — 0.5% drop chance — gold colour

DATABASE TABLES
Create these tables using Drizzle ORM:
items_master — master list of all items:
id, name, description, category (pill/recipe/treasure/artifact/ingredient), rarity, effect_type, effect_value, effect_duration, is_craftable, is_tradeable, is_chest_obtainable, icon, stack_limit
user_inventory — items owned by each user:
id, user_id, item_id, quantity, acquired_at
user_chests — unopened chests per user:
id, user_id, chest_type, quantity, earned_at
active_effects — currently active timed effects per user:
id, user_id, item_id, effect_type, effect_value, expires_at
crafting_recipes — all valid recipes:
id, result_item_id, ingredient_1_id, ingredient_2_id, ingredient_3_id, ingredient_4_id, required_cauldron, base_success_rate, is_discoverable
known_recipes — recipes each user has discovered:
id, user_id, recipe_id, discovered_at
failure_ashes — count of failure ashes per user:
id, user_id, count
permanent_modifiers — tracks all permanent XP bonuses per user:
id, user_id, source_item_id, modifier_type, modifier_value, applied_at
(Combined permanent XP bonus from all sources is hard-capped at +60% total. This cap is enforced server-side on every XP grant.)
karma_pill_log — tracks XP lost from crafting failures since last Karma Pill use:
id, user_id, xp_lost, lost_at
(Reset to zero when a Karma Pill is consumed. Only crafting failure events write to this table — pill use and other item effects do not.)
item_use_log — tracks item usage for rate-limiting and counter-based effects:
id, user_id, item_id, used_at

CHESTS
(Drop tables unchanged)
📦 Mortal Chest — earned every sprint completion:
55% Common, 30% Uncommon, 10% Rare, 5% Epic | Pills, Treasures, Ingredients only
🗃️ Iron Chest — earned by winning a sprint:
10% Uncommon, 50% Rare, 30% Epic, 10% Mythic | Pills, Treasures, Ingredients
💎 Crystal Chest — earned by completing any Gladiator match:
20% Rare, 45% Epic, 30% Mythic, 5% Legendary | Pills, Treasures, Artifacts, Ingredients
🔥 Inferno Chest — earned by 7 consecutive days of sprinting:
10% Epic, 55% Mythic, 35% Legendary | Pills, Treasures, Artifacts, Recipes
👑 Immortal Chest — earned by reaching a new rank:
Guaranteed one Mythic item plus 30% chance of one additional Legendary item | All categories
Rules:

Chests stack; open one at a time
Recipe scrolls only appear in Iron chest and above
Legendary items never appear in Mortal or Iron chests
Ingredients appear in all chest types
Maximum 5 Mortal Chests receivable per day from all sources combined
Unopened chests in overflow expire after 48 hours with a warning notification sent 24 hours before expiry


BAG SYSTEM

Default bag size is 20 slots
Pills, Treasures, Artifacts each take 1 slot per item
Ingredients stack up to 99 per slot
Recipe scrolls take 1 slot
Bag page has four tabs: All, Pills, Treasures & Artifacts, Ingredients, Recipes
Overflow is 5 slots maximum. Items in overflow for more than 48 hours are automatically discarded with a warning notification sent 24 hours before. Overflow must feel urgent — it is not bonus storage.
Bag expansion items automatically increase slot count permanently when used
Clicking any item shows full detail modal
Active timed effects show countdown timer
Items on cooldown show greyed Use button with time remaining


CRAFTING SYSTEM
1. Fusion tab

Select 3 identical items of the same name and rarity
Fuse into 1 item of the next rarity tier
100% success rate always
Works for Pills, Treasures, Artifacts, and Ingredients
Fusion chain: Common → Uncommon → Rare → Epic only. Fusion cannot produce Mythic or Legendary items. Reaching Mythic and Legendary requires Alchemy or Tribulation Crafting. This prevents any player from grinding their way to end-game items purely through fusion.

2. Alchemy tab

Drag 2–4 ingredients and items into slots
System detects if combination matches a known recipe
Shows predicted result and success rate before confirming
Success rates by cauldron: No cauldron 60% | Bronze 80% | Spirit 95% | Heaven 100% | Chaos 100% + can attempt Legendary recipes
1 hour cooldown between attempts (removed by Heaven Cauldron or Immortal Alchemy scroll)
Failed craft: all ingredients destroyed, user receives 1 Failure Ash; the XP value of destroyed ingredients is logged to karma_pill_log for the user
Collect 5 Failure Ashes → can craft random Common pill via Refining Furnace
Only recipes the user has learned appear (basic fusion always visible)

3. Tribulation Crafting tab

Legendary-only recipes using Mythic and Legendary ingredients
Requires recipe learned via Legendary Recipe Scroll
Success rates: Base 30% | Heaven Cauldron 50% | Chaos Cauldron 70% | Jade Emperor's Decree active 100%
24 hour cooldown between attempts
Failed tribulation outcomes (random):

40% — all ingredients lost, nothing gained
30% — ingredients lost, random Epic item given as consolation
30% — Tribulation Backlash — user loses XP equal to 10% of their current total XP (not a flat 500), but keeps all ingredients to retry. This makes Backlash genuinely painful at high ranks rather than a preferred outcome.



Special crafting mechanics

Pill Splitting (requires scroll): destroy 1 pill to receive exactly 1 pill of the rarity below — not 2. This is a pure downgrade, not a duplication mechanic.
Essence Extraction (requires scroll): destroy any non-ingredient item to get 1 Pure Essence
Pure Essence substitutes for any single ingredient in any recipe
Destroying a Legendary item gives Immortal Essence which substitutes for any ingredient AND boosts success rate by 20%


PERMANENT MODIFIER RULES
These rules govern all effects that grant permanent XP bonuses to prevent runaway compounding.

All permanent XP bonuses are tracked in the permanent_modifiers table
Sources: Undying Pill (+10%), Heaven Defying Constitution (+25%), Dao of Writing (up to +50%), Wheel of Reincarnation (×2 multiplier)
The combined additive permanent bonus from all sources is hard-capped at +60% sprint XP
The Wheel of Reincarnation ×2 multiplier is applied after the additive cap, but the total effective multiplier from all permanent sources combined cannot exceed ×3
Using the Wheel of Reincarnation resets all Dao of Writing stacks and all entries in permanent_modifiers to zero before applying the new ×2 multiplier
Permanent bonuses apply to sprint XP only — they do not multiply XP granted by pills or items
All permanent modifier logic is validated server-side only


CHEST EARNING TRIGGERS

Every sprint completion → +1 Mortal Chest (subject to 5/day cap)
Win a sprint (highest word count in room) → +1 Iron Chest
Complete any Gladiator match → +1 Crystal Chest
7 consecutive days with at least one sprint → +1 Inferno Chest
Reach a new rank → +1 Immortal Chest
Nuwa's Stone active → +1 Mortal Chest every 24 hours (subject to 5/day cap)

Permanent modifier cap enforced server-side on every XP grant — client never controls permanent bonus values
Items affecting combat are reserved for Kart mode only — do not implement here
